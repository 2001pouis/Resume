การเขียนสะท้อนตนเองแบบการเรียนรู้ผ่านประสบการณ์ Week # 1
(Recursive) ประสบการณ์ในสัปดาห์นี้มีการเชื่อมต่อกับประสบการณ์หรือแผนการที่ผ่านมาหรือไม่? (มี : ได้เรียนภาษา HTML CSS JS ในช่วงที่ฝึกงานกับสถานประกอบการ)
(CE) อะไรคือประสบการณ์ที่ผ่านมาของคุณ ? (ผมได้อยู่ทีมพี่เลี้ยงที่ทำเว็บไซต์ จึงได้ใช้ภาษHTML CSS และJS จึงได้ใช้อยู่เป็นประจำ)
(RO) อะไรคือประสบการณ์ที่สำเร็จและล้มเหลว ? (ความสำเร็จคือเวลาที่แก้ปัญหาของลูกค้าแต่ละอันได้ ล้มเหลวคือการที่ผมใช้เวลากับการแก้ไขนานเกินไป)
(AC) คุณได้เรียนรู้อะไรจากประสบการณ์ ? (จากประสบการณ์คือเราควรจะใช้มันบ่อยๆ จะทำให้เข้าใจและรู้ที่มาของมัน)
(AE) คุณจะได้นำสิ่งที่เรียนรู้จากประสบการณ์ ไปใช้อย่างไรต่อไป ? (จะไม่หยุดที่จะเรียนรู้อะไรใหม่ๆ และลงมือทำซ้ำๆ)
